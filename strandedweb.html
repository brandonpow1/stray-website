Frequently Asked Questions

It's not working! Help!

This version of Stranded runs on RenPyWeb which requires WebAssembly and Emscripten support. If you see the following error message:

...then you can't run Stranded on this browser. Sorry.

Currently the following browsers are confirmed not being able to work:

  • - iOS: any and all browsers FOR DEVICES OLDER THAN IOS 15.7 (they're all reskins of mobile Safari)
  • - Internet Explorer 11 (no WebAssembly support)

Though user-submitted testing, apparently iOS versions 15.7 and above have versions of Safari/Firefox etc. that can play Stranded (web edition). If you aren't yet updated to this version, we ask you to update iOS to at least version 15.7, as well as the browser of choice, and try again.
